# 任务：在白酒股中做一个回测

##### 白酒股股票代码包括： 000860, 002646, 600199, 603919, 000799, 603198, 600519, 600559, 002304, 000568, 600779, 600809, 000596, 603589, 600197, 603369, 000858

### 策略：每天尾盘卖出前一日买入的全部股票，再买入近10日涨幅最低的5个股票，平均分配资金。每天收盘换仓，不考虑交易成本，考虑停牌是顺延第六个股票。

### 目标：请计算一下，该策略'2020.12.31-2021.12.31'以及'2021.12.31-2022.6.30'的收益率分别是多少。

### 注意：要算复利，不能按单利算。

工具链路：
 -- 数据层面：使用akshare拿到数据，做必要的数据清洗和处理，包括：1、明确 AKShare 接口和复权方式 2、统一字段、日期、排序、类型 3、删除重复和明显异常数据  4、处理停牌/缺失交易日  5、明确“信号日”和“成交日”，避免未来函数。

 -- 策略层面：基于akquant.Strategy实现 mystrategy.

 -- 回测层面：利用akquant.run_backtest执行回测，分别计算'2020.12.31-2021.12.31'和'2021.12.31-2022.6.30'的收益率
